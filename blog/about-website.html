About this website

29 August 2022

For a long time I've been dreaming about creating some kind of an independent place for my art. There are often some changes in social media, and I wanted to have a static place, a place that could be unchanged until I decide to make any changes.

There are many different ways to create your personal website. I chose a very strange one - to write it myself. Of course, my knowledge of web programming is very basic. But making a website out of it is possible, so why not.

There is a portfolio of my favorite artworks, a page with a little information about me and this blog. A blog is a secondary thing on an artist's website, but I want to keep my drawings and some stories in chronological order too.

I'm so glad this site exists! Well, and my knowledge of HTML, CSS and JavaScript came in handy. Earlier I thought of becoming a programmer, but art has become very close to me, so I decided to do it while I have this opportunity.

So let there be many good pictures and interesting stories on this website! The beginning already exists.
